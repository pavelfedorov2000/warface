# About project

Warface gambling app

# Technologies stack:

- Create React App
- HTML5 (semantics, basic accessibility)
- React
- React Router DOM
- React Hooks
- Redux
- TypeScript
- classnames
- SCSS

# Concole commands:

- npm run server - start json-server at 3001 port
- npm start - start app at 3000 port
